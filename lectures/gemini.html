💡 Apps Script 간단 예제

Gmail에서 특정 키워드가 포함된 이메일을 자동으로 라벨링하는 스크립트:

function autoLabelEmails() {
  var threads = GmailApp.search('is:unread subject:"프로젝트"');
  var label = GmailApp.getUserLabelByName('프로젝트');
  
  for (var i = 0; i < threads.length; i++) {
    threads[i].addLabel(label);
    threads[i].markRead();
  }
}

이 스크립트를 트리거로 설정하면 매 시간마다 자동 실행됩니다.
